Exercice 1

Inscription des recrues

Écrivez un programme javascript qui vous permet de sauvegarder les informations d'une recrue. Les informations à sauvegarder sont :

  • le prénom
  • le nom de famille
  • le domaine d'étude
  • l'année de naissance

Ensuite, le programme devrait afficher le nom complet de la recrue, son domaine d'études et son âge (basé sur l'année de naissance seulement).
