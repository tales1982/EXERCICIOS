Convertisseur de mesure

Écrivez un programme javascript qui fonctionne comme un convertisseur de mesure.
Le programme doit demander une valeur en mètres,
puis donner la possibilité de choisir dans quelle unité de mesure cette valeur doit être convertie.
Les options sont :

  • millimètre (mm)
  • centimètre (cm)
  • décimètre (dm)
  • décamètre (dam)
  • hectomètre (hm)
  • kilomètre (km)

Le programme devrait alors convertir la mesure selon l'option choisie et afficher le résultat.
Le programme devrait également afficher un message "Option invalide"
si l'utilisateur entre une option autre que celles disponibles (utilisez break et default pour cela)
