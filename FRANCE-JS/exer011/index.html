À la recherche de palindromes

Écrivez un programme javascript capable d'identifier si un mot est un palindrome.
Un palindrome est un mot qui se lit à l'envers avec les mêmes lettres dans le même ordre.
Le programme devrait commencer à demander la saisie d'un mot, puis afficher un message
indiquant si ce mot est ou non un palindrome.
S'il ne s'agit pas d'un palindrome, le programme doit montrer le mot lu de gauche à droite et de droite à gauche.
